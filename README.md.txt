# Proyecto Requisición

Esta aplicación web permite cargar un archivo de Excel, procesarlo y generar un archivo de texto plano con un formato específico.

## Características
- Carga de archivos de Excel (.xls, .xlsx).
- Filtrado de datos según depósito.
- Generación de un archivo de texto plano estructurado.

## Tecnologías utilizadas
- **Frontend**: HTML, CSS, JavaScript.
- **Backend**: Node.js con Express.js.
- **Bibliotecas**:
  - `xlsx` para leer archivos de Excel.
  - `multer` para manejar la carga de archivos.
